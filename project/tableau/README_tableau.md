# Tableau Quick Guide

## Inputs
Use CSV folders generated by pipeline in `tableau/exports/`:
- `dashboard1_data_quality`
- `dashboard2_model_performance`
- `dashboard3_business_insights`
- `dashboard4_scalability`
- `lineage_log`

For a super-quick start, use:
- `data/samples/simple_crime_for_tableau.csv`

## Dashboards
1. Dashboard 1: data quality and pipeline monitoring
2. Dashboard 2: model performance and feature/business metric proxy
3. Dashboard 3: business insights and recommendations
4. Dashboard 4: scalability and cost analysis

## Best practices
- Convert each CSV input to Tableau Extract (.hyper).
- Add parameter controls for year and crime type.
- Add action filters from trend chart to detail table.
- Create mobile layout by hiding non-critical legends/labels.
